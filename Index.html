Тестовое задание для Sendsay
Прежде чем начать:
  • Измените переменные: password, login в MyTest.cpec.js на данные, которые вы будете вводить при авторизации на Sandsay
  • Так же измените переменные: name, mail, text
    • Name используется для отображения от чьего имени поступает рассылка
    • Mail используется для почты от который идёт рассылка
    • Text используется для текста, который будет отображаться в рассылке
Если вы проделали предыдущие шаги,то, пожалуйста, откройте консоль, перейдите в директорию данного проета и пропишите npm test
